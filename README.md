# To-do-list-
A website where you can list your activities and complete it on time 
 
Dark Mode To-Do List

A modern and interactive To-Do List web application built using HTML, CSS, and JavaScript.  
The application features a clean dark mode interface, colorful task cards, smooth hover animations, and localStorage support to persist tasks after page reload.

---

Features

Dark mode user interface  
Random color palette for each task  
Add new tasks  
Mark tasks as completed  
Delete individual tasks  
Clear all tasks at once  
Tasks saved using browser Local Storage  
Smooth hover and click animations  


## How It Works

- Users can add tasks through an input field.
- Each task is assigned a random color from a predefined palette.
- Clicking on a task toggles its completed state.
- Tasks are stored in localStorage to ensure persistence after page refresh.
- The Clear All button removes all tasks and clears stored data.
